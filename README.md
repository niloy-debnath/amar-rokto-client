## About the Project 
Amar Rokto is a community-driven blood donation platform that connects willing donors with recipients in need. Users can:

- Search for blood donors by blood group and location

- Register as a donor and provide personal & donation details
 
- View donor profiles with contact information
 
- Join donation events and track participation
 
- Communicate directly with donors or recipients
 
- Access a modern, mobile-friendly, and intuitive UI

The platform uses React, Firebase Authentication, and MongoDB to provide secure and real-time functionality.
---

## Project Overview  
This platform aims to save lives through blood donations by enabling users to:

- Discover nearby blood donors
 
- Contact donors quickly and securely
 
 Register as a donor and update donation history
- 
 View detailed donor profiles
 
- Ensure only authenticated users can access sensitive donor  information

- Benefit from a responsive, user-friendly interface



---

## Key Features  
- **Responsive Design:** Works on mobile, tablet, and desktop

- **User Authentication:** Firebase email/password login and Google login

- **Donor Profiles:** Detailed information including blood group, contact, age, last donation date

- **Search & Filter Donors:** By blood group and location

- **Be a Donor:** Form to register and update donor information

- **Private Routes:** Only logged-in users can view donor details or manage profiles

- **Dynamic Routing:** Individual donor pages with unique URLs

- **Modern UI/UX:** Tailwind CSS + DaisyUI + Framer Motion animations

- **Real-Time Updates:** Changes reflected immediately using Firebase and MongoDB
